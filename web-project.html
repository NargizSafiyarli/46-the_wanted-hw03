How Websites Work?

Our team, "The Wanted," consists of dedicated members who worked collaboratively on various aspects of the project including our team leader Nargiz Safiyarli, Bahman Sariyev and Rafael Heybatzada.Nargiz was responsible for developing and designing the Home page (index.html), the "Our Team" page (team-members.html), and the "Coding Event" page (hour-of-code.html). She utilized HTML and CSS to create the navigation bar, structure the pages, and design layouts. Additionally, she developed interactive elements for the website using JavaScript (script.js).Bahman focused on developing and designing the "Project Details" page (website-project.html) and the "Code Artist" page (code-artist.html). He implemented HTML, CSS, and JavaScript to create designs and integrate code art examples with a minimalist approach.Rafael played a key role in the creative and content development aspects. He came up with the team name, "The Wanted," and designed the team logo. He wrote all the textual content for the website, including sections about the team, team members, and the "Hour of Code" event. Additionally, Rafael was responsible for writing the project report.

Features of Our Website

Feature Description
Home Page The homepage introduces our team, "The Wanted," and provides an easy-to-navigate interface. It includes a stylish navigation bar and interactive elements to enhance the user experience.
Our Team This section contains detailed information about each team member, their skills, and their roles in the project. It highlights our collaborative efforts and contributions. You can also find each member’s email addresses for communication.
Hour of Code Event A dedicated page sharing insights into our participation in the Hour of Code, showcasing how we applied coding skills to meaningful projects and events.In addition, this page include a video, photos and a short report about our team.
Project Details This section provides an in-depth look at the goals and outcomes of our website project. It includes links to our individual Codecademy profiles, where visitors can track our coding progress.
Code Art A creative page featuring examples of algorithmic art, reflecting the innovative use of programming for artistic purposes.
Clock Our website includes a live clock feature, displaying the current time dynamically.
